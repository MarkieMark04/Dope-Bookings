Capacity Rules

How It Works

Confirmed bookings fill the main capacity first.
Overbooking opens only after confirmed spots are full.
Waitlist starts after confirmed and overbooking spots are exhausted.
Auto-close blocks new bookings when no configured spots remain.

Where To Set It

Session Controls

Create or edit a class/event to set waitlist, overbooking, and auto-close rules.
Each session keeps its own capacity settings.
Attendee lists show confirmed, overbooked, and waitlisted statuses.
